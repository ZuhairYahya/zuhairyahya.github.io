Install PhpMyAdmin

Thu, Apr 22, 2021 One-minute read

Pada artikel ini, saya akan menjelaskan cara menginstall PhpMyAdmin pada Ubuntu 18.04

Install PhpMyAdmin

  1. Update repository
sudo apt-get update
  1. Install apache2
sudo apt-get install apache2
  1. Jika berhasil, silahkan cek pada browser
http://ip-address
  1. Install mysql server
sudo apt-get install mysql-server
  1. Lakukan authentication, masukkan username dan password
sudo mysql -u root -p

CREATE USER 'username'@'localhost' IDENTIFIED BY 'password';
GRANT ALL PRIVILEGES ON *.* TO 'username'@'localhost' WITH GRANT OPTION;
exit;
  1. Install library
sudo apt-get install php php-mysql libapache2-mod-php php-cli php-cgi php-gd mysql-server mysql-client zip -y
  1. Masuk directory html
cd /var/www/html
  1. Berikan akses untuk info.php
sudo touch info.php
  1. Masukkan perintah berikut
sudo nano info.php

<?php 

phpinfo(); 

?>
  1. Buka browser untuk mengecek info.php
http://IP-Server/info.php
  1. Install phpmyadmin
sudo apt-get install phpmyadmin
  1. Cek browser untuk membuka PhpMyAdmin
http://IP-server/phpmyadmin

Semoga bermanfaat untuk anda.
