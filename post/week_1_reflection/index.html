Weekly Reflection 1

Sat, Jul 10, 2021 2-minute read

Hallo nama saya Zuhair Yahya, saat ini saya sedang mengikuti program Generasi GIGIH dari Yayasan Anak Bangsa Bisa (YABB) untuk learning path Backend Engineer. Pada weekly reflection 1 ini saya akan menjelaskan beberapa hal mengenai apa yang sudah saya pelajari selama 1 minggu ini pada modul 1 untuk membahas mengenai Object Oriented Programming (OOP) dengan menggunakan bahasa pemrograman Ruby.

Saya merasakan selama 1 minggu pembelajaran ini cukup banyak perubahan yang signifikan, karena sebelumnya saya sama sekali belum memahami mengenai konsep dari Object Oriented Programming itu sendiri seperti apa. Dengan penjelasan selama dikelas, dengan melalui homework yang diberikan setiap selesai kelas dan juga dengan belajar sendiri dan dibantu oleh teman membuat saya mudah dalam memahami konsep dari Object Oriented Programming ini dan bagaimana proses implementasi dari konsep Object Oriented Programming itu sendiri. Seperti memahami mengenai Abstraction, Encapsulation, Inheritance dan Polymorphism itu sendiri dengan mengimplementasikan pada program game dan selama 1 minggu itu dengan mengerjakan homework tersebut sangat membantu saya yang sebenarnya basic saya dari SysAdmin atau Ops Engineer yang jarang bersentuhan dengan programming. Namun dengan 1 minggu untuk mengerjakan homework dan diskusi dikelas itu membantu saya untuk memahaminya.

Hal paling penting menurut saya adalah dalam memahami sebuah konsep dari Obejct Oriented Programming itu sendiri karena ketika nantinya akan membuat sebuah program. Konsep OOP lah yang akan digunakan dan kita memahami konsep tersebut, maka akan mudah dalam melakukan troubleshooting ketika mengalami error karena konsep OOP sendiri menurut saya adalah sebuah fundamental dalam belajar sebuah programming. Seperti halnya dalam membuat sebuah class dan konsep pewarisan atau inheritance itu harus paham karena itu membantu dalam membuat sebuah program.

Hal yang kurang saya mengerti untuk konsep Polymorphism, saya belum memahami 100% mengenai konsepnya dan implementasi dalam sebuah program tersebut seperti apa. Namun saya mencoba mehaminya dengan belajar mandiri atau saya berdiskusi dengan teman saya yang memang sudah memahami konsep tersebut dan kebetulan teman saya memang bekerja sebagai Backend Engineer. Hal tersebut yang dapat saya lakukan untuk mengatasinya karena memang dalam melakukan coding hal yang harus dilakukan adalah dengan praktek dan sering-sering menemukan pesan error. Dengan sering praktek dan menemukan pesan error maka logika dan problem solving tersebut akan terus terasah dan terbiasa.

Namun sebagain besar saya sudah mulai memahaminya dan sudah mengerti bagaimana cara membuat program dengan menggunakan konsep Object Oriented Programming (OOP) dengan menggunakan bahasa Ruby. Semoga pada modul kedua nanti saya menemukan hal-hal baru yang belum pernah saya pelajari sebelumnya. Terimakasih Generasi GIGIH
