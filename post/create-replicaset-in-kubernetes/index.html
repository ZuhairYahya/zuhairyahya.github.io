Create ReplicaSet In Kubernetes

Sun, Aug 29, 2021 One-minute read

Pada artikel ini saya akan menjelaskan bagaimana cara untuk membuat replicaset pada kubernetes cluster

Create ReplicaSet

  1. Scale Pod dengan terminal
kubectl scale --replicas=3 deployment nginx
  1. Ubah replicaset
kubectl scale --replicas=1 deployment nginx
  1. Membuat replicaset menggunakan file deployment replica.yaml
apiVersion: apps/v1
kind: ReplicaSet
metadata:
  name: nginx
spec:
  replicas: 2
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      name: nginx
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx
        ports:
        - containerPort: 80
  1. Jalankan file deployment
kubectl apply -f replica.yaml
