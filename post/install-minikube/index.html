Install Minikube

Mon, Feb 15, 2021 One-minute read

Pada artikel pertama ini, akan dijelaskan mengenai cara menginstall minikube di Linux

Install Minikube

  1. Install minikube menggunakan unduhan langsung
curl -Lo minikube https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64
  1. Jadikan biner minikube dapat dieksekusi
chmod +x minikube
  1. Tambahkan program minikube kedalam PATH anda
sudo mkdir -p /usr/local/bin/
sudo install minikube /usr/local/bin/
  1. Jika anda belum mempunyai docker grup, silahkan anda tambahkan
sudo groupadd docker
  1. Jika anda belum mempunyai sebuah user, silahkan buat
sudo adduser $USER
  1. Anda dapat menambahkan pengguna yang telah dibuat ke docker grup
sudo usermod -aG docker $USER
  1. Silahkan Logout dan jalankan images tanpa menggunakan parameter sudo
docker run hello-world
  1. Silahkan jalankan minikube
minikube start
  1. Silahkan cek status minikube
minikube status

Semoga bermanfaat
