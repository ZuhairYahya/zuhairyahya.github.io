Weekly Reflection 2

Sun, Jul 18, 2021 2-minute read

Halo nama saya Zuhair Yahya, saya dari learning path Backend Engineer. Pada Weekly 2 Reflection ini saya akan sedikit bercerita hasil pembelajaran saya selama minggu kedua ini bersama dengan Generasi GIGIH dari Yayasan Anak Bangsa Bisa. Pada minggu kedua ini saya mempelajari mengenai HTPP Method, Routing and Rendering pada sesi pertama. Sebelumnya saya belum banyak mengetahui megenai materi tersebut terlebih karena memang dibangku perkuliahan juga tidak membahas mengenai hal tersebut. Namun, mengenai HTTP Method saya pernah mempelajarinya secara otodidak, hal tersebut yang sedikit membantu saya untuk dapat memahaminya ketika dikelas sesi pertama. Namun, saya menemukan hal baru mengenai Routing dan Rendering dengan menggunakan library dari Ruby yaitu Sinatra untuk menjalankan API pada website dan pada sesi pertama saya banyak menemukan dan belajar hal baru. Selanjutnya, pada sesi kedua saya juga belum pernah mempelajari sebuah database, walaupun ketika semester 6 lalu saya sempat bekerja sebagai Cloud Engineer dan membuat database untuk kebutuhan monitoring Grafana dengan menggunakan InfluxDB tetapi ternyata saya banyak mengetahui banyak hal mengenai Query pada database dengan menggunakan MySQL dan pada sesi ketiga saya juga mempelajari hal baru mengenai untuk mengALTER sebuah table dan banyak lainnya yang akhirnya saya menemukan keasikan tersendiri.

Perasaan saya sangat senang dan sungguh sangat penasaran dengan tugas akhir yang diberikan, bagian yang paling saya suka adalah ketika membuat sebuah database dari awal yaitu membuat sebuah design dan memberikan entity pada sebuah database lalu menggabungkan dari berbagai tabel menjadi satu. Ketika hal tersebut berhasil saya buat, saya menemukan kesenangan tersendiri karena ketika mendesign sebuah database diawal itu seakan saya mempunyai hak akses penuh terhadap sebuah tabel dan data yang akan saya buat, dan tipe data apa yang akan saya gunakan juga itu memberikan saya kebebasan tersendiri untuk mengeksplornya.

Dengan pengetahuan yang sudah saya pelajari minggu ini, kedepannya saya akan mempelajarinya lebih dalam mungkin bisa menggunakan MongoDB atau database lainnya dan saya akan coba untuk menggunakannya dalam pekerjaan saya jika memang suatu saat nanti saya bekerja yang mengharuskan saya untuk mendesign atau membuat sebuah database. Ilmu yang saya dapat ketika mengikuti kelas ini tentunya menjadi sebuah fondasi saya dan akan saya praktekan nantinya untuk menunjang kebutuhan pekerjaan saya. Karena menurut saya ilmu yang saya dapat dikelas ini merupakan sebuah fondasi awal, namun saya juga harus tetap mengeksplornya lebih jauh karena saya percaya bahwa perkembangan teknologi terus berkembang.
