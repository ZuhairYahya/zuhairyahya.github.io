Install Grafana

Tue, Feb 23, 2021 One-minute read

Pada artikel ketiga ini akan dibahas mengenai cara menginstall grafana menggunakan centos 7 untuk kebutuhan monitoring

Install Grafana

  1. Tambahkan Grafana RPM repository ke dalam sistem
sudo tee  /etc/yum.repos.d/grafana.repo<<EOF
[grafana]
name=grafana
baseurl=https://packages.grafana.com/oss/rpm
repo_gpgcheck=1
enabled=1
gpgcheck=1
gpgkey=https://packages.grafana.com/gpg.key
sslverify=1
sslcacert=/etc/pki/tls/certs/ca-bundle.crt
EOF
  1. Silahkan buat repo
vi /etc/yum.repos.d/grafana.repo

[grafana]
name=grafana
baseurl=https://packages.grafana.com/oss/rpm
repo_gpgcheck=1
enabled=1
gpgcheck=1
gpgkey=https://packages.grafana.com/gpg.key
sslverify=1
sslcacert=/etc/pki/tls/certs/ca-bundle.crt
  1. Install Grafana
yum install grafana
  1. Reload daemon
systemctl daemon-reload
  1. Start Grafana
systemctl start grafana-server
  1. Cek status grafana
systemctl status grafana-server
  1. Enable grafana
systemctl enable grafana-server
  1. Buka port firewall tcp 3000
iptables -A INPUT -p tcp --dport 3000 -j ACCEPT
  1. Install iptables
yum install iptables-services
  1. Start iptables
systemctl start iptables
  1. Enable iptables
systemctl enable iptables
  1. Cek status iptables
systemctl status iptables
  1. Simpan iptables
service iptables save
  1. Cek menggunakan web browser menggunakan url
http://Ipserver_grafana:3000/
  1. Masuk admin
Username : admin
Password : admin

Mungkin itu saja yang bisa dijelaskan, semoga bermanfaat
