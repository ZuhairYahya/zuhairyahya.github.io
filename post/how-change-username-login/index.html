How To Change Username Login In Ubuntu

Sun, Aug 29, 2021 One-minute read

Pada artikel ini saya akan membahas mengenai cara merubah username login pada ubuntu

Change Username

  1. Set password root
sudo passwd root
  1. Ubah hostname
sudo hostnamectl set-hostname [new-hostname]

  1. Login root

  2. Ubah username dan folder direktory

usermod -l [new-username] -d /home/[new-username] -m [old-username]
  1. Ubah nama grup
groupmod -n [new-username] [old-username]
  1. Kunci akun root
passwd -l root
  1. Logout
exit
