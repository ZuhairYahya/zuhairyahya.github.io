How To Setup IP Static In Ubuntu

Thu, Sep 16, 2021 One-minute read

Pada artikel ini saya akan coba menjelaskan bagaimana cara mengkonfigurasi ip static pada Centos

Setting IP Static Di Centos

  1. Buka file konfigurasi
$ sudo vi /etc/sysconfig/network-scripts/ifcfg-enp0s3
  1. Ubah script menjadi
TYPE="Ethernet"
PROXY_METHOD="none"
BROWSER_ONLY="no"
BOOTPROTO="static"
DEFROUTE="yes"
IPV4_FAILURE_FATAL="no"
IPV6INIT="yes"
IPV6_AUTOCONF="yes"
IPV6_DEFROUTE="yes"
IPV6_FAILURE_FATAL="no"
IPV6_ADDR_GEN_MODE="stable-privacy"
NAME="enp0s3"
UUID="ae298adb-1b94-42a0-9dc9-a121c7561a5b"
DEVICE="enp0s3"
ONBOOT="yes"
IPADDR=192.168.1.2
NETMASK=255.255.255.0
GATEWAY=192.168.1.1
DNS1=8.8.8.8
DNS2=8.8.4.4
  1. Restart Network
$ sudo service network restart
  1. Cek ip address
$ ip a
