Install Nginx Using Ansible

Mon, Nov 15, 2021 One-minute read

Halo pada artikel ini saya akan menjelaskan cara menginstall nginx menggunakan ansible sebagai automation dengan menggunakan 2 server ubuntu

Install Nginx Using Ansible

  1. Create inventory file
vi inventory

[webserver]
ansible1
ansible2
  1. Create ansible cfg file
vi ansible.cfg

[defaults]
inventory = ./inventory
  1. Create ansible playbook
vi nginx.yml

- name: Install and Start Nginx Web Server
  hosts: webserver
  become: true
  tasks:
    - name: Install nginx
      apt:
        name: nginx
        update_cache: yes
        state: present

    - name: Start nginx
      apt:
        name: nginx
        state: started
        enabled: true
  1. Check list hosts
ansible all --list-hosts
  1. Run ansible playbook
ansible-playbook nginx.yml
  1. Check status nginx
sudo systemctl status nginx
