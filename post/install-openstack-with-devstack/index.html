Install Openstack With Devstack

Wed, Apr 7, 2021 One-minute read

Pada artikel keenam ini, saya akan mencoba untuk menjelaskan mengenai cara menginstall Openstack pada ubuntu 18.04 dengan Devstack

Install Openstack Devstack

  1. Tambahkan ip pada hosts
sudo vi /etc/hosts
  1. Update packages
sudo apt update
  1. Buat user bernama stack
sudo useradd -s /bin/bash -d /opt/stack -m stack
  1. Mengaktifkan sudo tanpa menggunakan password
echo "stack ALL=(ALL) NOPASSWD: ALL" | sudo tee /etc/sudoers.d/stack
  1. Masuk sebagai user stack
sudo su - stack
  1. Install Git
sudo apt -y install git
  1. Clone devstack
git clone https://opendev.org/openstack/devstack
  1. Masuk directory devstack
cd devstack
  1. Buat file local.conf
sudo nano local.conf

[[local|localrc]]
# Password untuk KeyStone, Database, RabbitMQ dan Service
ADMIN_PASSWORD=(sesusaikan)
DATABASE_PASSWORD=$ADMIN_PASSWORD
RABBIT_PASSWORD=$ADMIN_PASSWORD
SERVICE_PASSWORD=$ADMIN_PASSWORD
# Host IP - IP VM yang telah dikonfigurasi sebelumnya (cek : ifconfig)
HOST_IP=(ip address sesuaikan)
  1. Deploy openstack devstack
./stack.sh
  1. Jika sudah akses melalui browser
http://ip-address/dashboard

Semoga bermanfaat untuk anda
