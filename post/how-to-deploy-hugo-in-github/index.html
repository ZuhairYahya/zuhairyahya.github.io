How To Deploy Hugo In Github

Sun, Aug 29, 2021 One-minute read

Pada artikel ini saya akan menjelaskan bagaimana cara mendeploy hugo pada akun github sebagai tempat penyimpanan

Create New Hugo

  1. Buatlah web hugo
hugo new site website
  1. Install Tema
cd themes
git clone https://github.com/xianmin/hugo-theme-jane.git --depth=1 themes/jane >>> (contoh)
  1. Edit file config.toml dan sesuaikan tema
sudo config.toml
  1. Jalankan hugo server
hugo server

Deploy Hugo

  1. Buatlah dua buah repositori di github
website (bebas) >> repositori 1
user.github.io >> repositori 2

  1. Pada repository (website) digunakan untuk menyimpan file hugo

  2. Pada repository (user.github.io) digunakan untuk menyimpan file public

  3. Siapkan repositori lokal

cd website

git init

rm -rf public

git remote add origin https://github.com/{username}/{nama-repo}.git
  1. Push ke github
git add .

git commit -m "first commit"

git push origin master
  1. Buatlah submodule pada repositori github.io
git submodule add -b master https://github.com/{username}/{username}.github.io.git public
  1. Push public ke github
hugo >> menarik file static website

cd public

git add .

git commit -m "first commit"

git push -u origin master

Mungkin itu saja yang dapat saya jelaskan. Terimakasih
