Install Cpanel WHM

Wed, Apr 21, 2021 One-minute read

Pada artikel ini, saya akan menjelaskan cara menginstall CPanel WHM pada Centos 7

Install CPanel

  1. Update repository
yum update
  1. Install perl
yum install perl -y
  1. Install wget
yum install wget -y
  1. Install curl
yum install curl -y
  1. Download script installer CPanel menggunakan curl
curl -o latest -L https://securedownloads.cpanel.net/latest
  1. set hostname menggunakan nmtui dan masukkan hostname anda
nmtui

hostname = host.your-domain.com
  1. Restart systemd
systemctl restart systemd-hostnamed
  1. Stop network manager service
systemctl stop NetworkManager.service
  1. Disable network manager service
systemctl disable NetworkManager.service
  1. Jalankan script installer CPanel
sh latest
  1. Akses CPanel menggunakan browser
http://ip-address:2087
