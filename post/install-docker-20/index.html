Install Docker Ubuntu 20.04

Thu, Oct 21, 2021 One-minute read

Pada artikel ini saya akan membahas bagaimana cara untuk menginstall docker pada ubuntu 20.04

Install Docker

  1. Update packages
$ sudo apt-get update
  1. Install Dependencies
$ sudo apt-get install apt-transport-https ca-certificates curl gnupg-agent software-properties-common
  1. Add Docker GPG Key
$ sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
  1. Install Docker Repository
$ sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu focal stable"
  1. Update packages
$ sudo apt-get update
  1. Install Docker CE
$ sudo apt-get install docker-ce docker-ce-cli containerd.io
  1. Cek status Docker
$ sudo systemctl status docker
  1. Cek versi Docker
$ docker version
  1. Cek detail informasi
$ sudo docker info
  1. Jalankan image
$ sudo docker run hello-world
  1. Menampilkan image
$ sudo docker image ls
  1. Delete image
$ sudo docker image rm -f hello-world
