Install InfluxDB

Tue, Feb 23, 2021 One-minute read

Pada artikel keempat ini akan menjelaskan mengenai cara menginstall influxdb

Install InfluxDB

  1. Tambahkan influxdb kedalam repository
cat <<EOF | sudo tee /etc/yum.repos.d/influxdb.repo
[influxdb]
name = InfluxDB Repository - RHEL \$releasever
baseurl = https://repos.influxdata.com/rhel/\$releasever/\$basearch/stable
enabled = 1
gpgcheck = 1
gpgkey = https://repos.influxdata.com/influxdb.key
EOF
  1. Update cache
yum makecache fast
  1. Install influxdb
yum -y influxdb vim curl
  1. Start influxdb
systemctl start influxdb
  1. Enable influxdb
systemctl enable influxdb
  1. Buka port firewall tcp 8086
iptables -A INPUT -p tcp --dport 8086 -j ACCEPT
  1. Install iptables
yum install iptables-services
  1. Start iptables
systemctl start iptables
  1. Enable iptables
systemctl enable iptables
  1. Cek status iptables
systemctl status iptables
  1. Simpan iptables
service iptables save

Mungkin itu saja yang bisa jelaskan, semoga bermanfaat untuk anda
